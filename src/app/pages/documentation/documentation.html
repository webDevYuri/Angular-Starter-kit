Toast Notifications

Simple, non-blocking alerts that appear at the top of the screen

Available Methods

toastService.success(message)

Display a success notification

toastService.error(message)

Display an error notification

toastService.warning(message)

Display a warning notification

toastService.info(message)

Display an info notification

Example Usage

import { ToastService } from './core/services/toast.service';

export class MyComponent {
  constructor(private toastService: ToastService) {}

  handleSuccess() {
    this.toastService.success('Operation successful!');
  }

  handleError() {
    this.toastService.error('Something went wrong');
  }
}

Confirmation Modal

Blocking dialogs for important user confirmations

Modal Types

danger

Red colored, for destructive actions like delete

warning

Amber colored, for important cautionary actions

info

Blue colored, for informational confirmations

Example Usage

import { ConfirmService } from './core/services/confirm.service';

export class MyComponent {
  constructor(private confirmService: ConfirmService) {}

  async deleteItem() {
    const confirmed = await this.confirmService.confirm({
      title: 'Delete Item',
      message: 'This action cannot be undone. Continue?',
      confirmText: 'Delete',
      cancelText: 'Cancel',
      type: 'danger'
    });

    if (confirmed) {
      // User confirmed
    }
  }
}

AOS Animations

Scroll-based animations using the Animate On Scroll library

Available Animations

fade-up

Fade in from below

fade-down

Fade in from above

fade-left

Fade in from right

fade-right

Fade in from left

zoom-in

Scale up into view

flip-up

Flip into view

Usage in Templates

<div data-aos="fade-up">
  Animates on scroll
</div>

<!-- With delay -->
<div data-aos="fade-up" data-aos-delay="200">
  Delayed animation
</div>

<!-- With custom duration -->
<div data-aos="fade-up" data-aos-duration="800">
  Slower animation
</div>

Global Configuration

AOS is initialized in app.ts with these defaults:

import AOS from 'aos';

AOS.init({
  duration: 600,    // Animation duration in ms
  easing: 'ease-out', // Easing function
  once: true        // Animate only once
});

Dynamic Content

When elements are added/removed dynamically (e.g. with *ngIf), call AOS.refresh() to detect new elements:

import AOS from 'aos';

toggleContent() {
  this.showContent = !this.showContent;
  setTimeout(() => AOS.refresh(), 50);
}
